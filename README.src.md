<h2>About The Project</h2><!--en-->
<!--en-->
<img width="1920" height="1100" alt="image" src="https://github.com/user-attachments/assets/17399995-7032-4d90-957e-5cef278ceb6e" /><!--en-->
<img width="1920" height="1140" alt="image" src="https://github.com/user-attachments/assets/1f8cd087-a43a-45b8-a8f3-36c190d6e66c" /><!--en-->
<img width="1919" height="886" alt="image" src="https://github.com/user-attachments/assets/023274ee-b206-458b-a4af-d5c6e0950180" /><!--en-->
<!--en-->
Aura Text is an excellent text/code editor that offers a wide array of essential tools. Built with PyQt6 and Python, it harnesses the power and flexibility of these technologies.<!--en-->
With Aura Text, users can access a versatile and powerful editing environment. Whether working on a small script or a complex project, Aura Text equips you with all the necessary features to streamline your workflow. From syntax highlighting and code completion to smart indentation and advanced terminal, Aura Text ensures that your coding experience is efficient and enjoyable while being extremely light on your PC.<!--en-->
<!--en-->
<br><!--en-->
<!--en-->
***The main highlights of Aura Text are:***<!--en-->
- Edit files (duh)<!--en-->
- Support up to 30 languages<!--en-->
- Autocompletion with intelligent quote and string matcher<!--en-->
- Code Linting for Python (BETA) (More langs coming soon)<!--en-->
- Git Clone, Commit and Push with interactive rebase and graphs<!--en-->
- Command Palette for quick execution of tasks<!--en-->
- Custom title bars<!--en-->
- Regex Playground<!--en-->
- Split pane Markdown editor<!--en-->
- Terminal with history<!--en-->
- Python Console<!--en-->
- Plugin support<!--en-->
- Extensive theming including Material Theming support<!--en-->
- Extremely customisable (like literally you can change every aspect of it)<!--en-->
  <!--en-->

## Getting Started<!--en-->
<!--en-->
Let's set up Aura Text on your PC!<!--en-->
<!--en-->
### Prerequisites<!--en-->
- Windows 10 x64 or later, a Linux distro running kernel 6.x or later, or macOS Ventura or later <!--en-->
- Python 3.9 or later<!--en-->
- Python installation is bootstrapped with pip<!--en-->
- (Recommended) A fresh venv created with `python -m venv venv` and activated with `venv\Scripts\activate`<!--en-->
- The contents of `requirements.txt` installed via `pip install -r requirements.txt`<!--en-->
- (If building an installer) Inno Setup 6.4.3 or later<!--en-->
  <!--en-->
### Installation<!--en-->
<!--en-->
You can download a prebuilt installer from the Releases or build one yourself. If using prebuilt installers, just skip to the use section.<!--en-->
<!--en-->
#### Building the installer<!--en-->
<!--en-->
1. Clone the repo or download a tarball<!--en-->
2. Install all prerequisites<!--en-->
3. `python build.py` to compile the program first<!--en-->
4. Open up the `.iss` Inno Setup script and compile it via Ctrl+F9 or `Build > Compile` - installer can be found in `Output` folder<!--en-->
<!--en-->
<!--en-->
##### Using the installer<!--en-->
<!--en-->
Just run the `.exe` file, duh.<!--en-->
<!--en-->
### Testing<!--en-->
<!--en-->
This is for people who solely just want to run without installation for mostly testing purposes.<!--en-->
We need the prerequisites above. After getting them, you can run the program with `pythonw main.py` to run it without flooding your terminal with logging, or you can just run with `python main.py` to troubleshoot errors and debug it.<!--en-->
<!--en-->

- Windows 10 x64 ou posterior, distro Linux com kernel 6.x ou posterior, ou macOS Ventura ou posterior<!--pt-->
- Python 3.9 ou posterior<!--pt-->
- (Recomendado) Crie e ative um venv: `python -m venv venv` e `venv\Scripts\activate`<!--pt-->
- Instale dependências: `pip install -r requirements.txt`<!--pt-->
<!--pt-->
### Instalação<!--pt-->
<!--pt-->
Você pode baixar um instalador pré-compilado dos Releases ou construir um localmente.<!--pt-->
<!--pt-->
#### Construir o instalador<!--pt-->
1. Clone o repo ou baixe o tarball<!--pt-->
2. Instale todos os pré-requisitos<!--pt-->
3. Execute `python build.py` para compilar<!--pt-->
4. Abra o script Inno Setup `.iss` e compile com Ctrl+F9 ou `Build > Compile` — o instalador estará em `Output`<!--pt-->
<!--pt-->
##### Usar o instalador<!--pt-->
<!--pt-->
Execute o `.exe` gerado.<!--pt-->
<!--pt-->
### Testes<!--pt-->
<!--pt-->
Para testes: `pythonw main.py` (sem logs) ou `python main.py` (para debug)<!--pt-->
