Aura Text என்பது அத்தியாவசிய கருவிகளின் பரந்த வரிசையை வழங்கும் ஒரு சிறந்த உரை/குறியீடு திருத்தி. PyQt6 மற்றும் Python கொண்டு கட்டமைக்கப்பட்டது, இந்த தொழில்நுட்பங்களின் சக்தி மற்றும் நெகிழ்வுத்தன்மையை பயன்படுத்துகிறது.
Aura Text மூலம், பயனர்கள் பல்துறை மற்றும் சக்திவாய்ந்த திருத்தல் சூழலை அணுகலாம். சிறிய ஸ்கிரிப்டில் பணிபுரிந்தாலும் சரி, சிக்கலான திட்டத்தில் பணிபுரிந்தாலும் சரி, Aura Text உங்கள் பணிப்பாய்வை நெறிப்படுத்த தேவையான அனைத்து அம்சங்களையும் வழங்குகிறது. தொடரியல் முன்னிலைப்படுத்தல் மற்றும் குறியீடு தானியங்கி நிரப்பல் முதல் ஸ்மார்ட் உள்தள்ளல் மற்றும் மேம்பட்ட முனையம் வரை, Aura Text உங்கள் PC-யில் மிகவும் இலகுவாக இருக்கும்போது திறமையான மற்றும் மகிழ்ச்சியான குறியிடல் அனுபவத்தை உறுதி செய்கிறது.

## 🏃 தொடங்குதல்

உங்கள் PC-யில் Aura Text-ஐ அமைப்போம்!

### முன்நிபந்தனைகள்
- Windows 10 x64 அல்லது அதற்கு பிற்பட்டது
- Python 3.9 அல்லது அதற்கு பிற்பட்டது
- Python நிறுவல் pip மூலம் bootstrap செய்யப்பட்டுள்ளது
- (பரிந்துரைக்கப்படுகிறது) `python -m venv venv` மூலம் புதிய venv உருவாக்கி `venv\Scripts\activate` மூலம் செயல்படுத்தவும்
- `pip install -r requirements.txt` வழியாக `requirements.txt`-இன் உள்ளடக்கங்களை நிறுவவும்
- (நிறுவி கட்டமைக்க வேண்டும் என்றால்) Inno Setup 6.4.3 அல்லது அதற்கு பிற்பட்டது
  
### நிறுவல்

Releases பகுதியிலிருந்து முன்கட்டப்பட்ட நிறுவியை பதிவிறக்கலாம் அல்லது நீங்களே கட்டமைக்கலாம்.

#### நிறுவியை கட்டமைத்தல்

1. களஞ்சியத்தை clone செய்யவும் அல்லது tarball பதிவிறக்கவும்
2. அனைத்து முன்நிபந்தனைகளையும் நிறுவவும்
3. முதலில் நிரலை தொகுக்க `python build.py` இயக்கவும்
4. `.iss` Inno Setup script-ஐ திறந்து Ctrl+F9 அல்லது `Build > Compile` மூலம் தொகுக்கவும் — நிறுவி `Output` கோப்புறையில் காணலாம்

##### நிறுவியை பயன்படுத்துதல்

`.exe` கோப்பை இயக்கவும்.

### சோதனை

முக்கியமாக சோதனை நோக்கங்களுக்காக நிறுவல் இல்லாமல் இயக்க விரும்புபவர்களுக்கு இது.
முனையத்தில் logging இல்லாமல் இயக்க `pythonw main.py` பயன்படுத்தவும், அல்லது பிழைத்திருத்தம் செய்ய `python main.py` பயன்படுத்தவும்.

## 🛂 பங்களிப்பு

பங்களிப்புகள்தான் திறந்த மூலக் குழுமத்தை கற்றுக்கொள்ள, தூண்டிட மற்றும் உருவாக்கிட ஒரு அற்புதமான இடமாக மாற்றுகின்றன. நீங்கள் செய்யும் எந்தப் பங்களிப்பும் **மிகவும் பாராட்டப்படுகிறது**.
இதை மேம்படுத்தும் ஒரு பரிந்துரை இருந்தால், களஞ்சியத்தை fork செய்து pull request உருவாக்கவும். "enhancement" tag-உடன் ஒரு issue-ஐயும் திறக்கலாம்.
திட்டத்திற்கு நட்சத்திரம் கொடுக்க மறக்காதீர்கள்! மீண்டும் நன்றி!

1. திட்டத்தை Fork செய்யவும்
2. உங்கள் அம்சக் கிளையை உருவாக்கவும் (`git checkout -b feature/AmazingFeature`)
3. உங்கள் மாற்றங்களை Commit செய்யவும் (`git commit -m 'Add some AmazingFeature'`)
4. கிளைக்கு Push செய்யவும் (`git push origin feature/AmazingFeature`)
5. Pull Request திறக்கவும்
