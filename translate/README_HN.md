Aura Text एक उत्कृष्ट टेक्स्ट/कोड संपादक है जो आवश्यक टूल्स की एक विस्तृत श्रृंखला प्रदान करता है। PyQt6 और Python के साथ निर्मित, यह इन तकनीकों की शक्ति और लचीलेपन का उपयोग करता है।
Aura Text के साथ, उपयोगकर्ता एक बहुमुखी और शक्तिशाली संपादन वातावरण तक पहुँच सकते हैं। चाहे आप एक छोटी स्क्रिप्ट पर काम कर रहे हों या एक जटिल प्रोजेक्ट पर, Aura Text आपको अपने वर्कफ़्लो को सुव्यवस्थित करने के लिए सभी आवश्यक सुविधाएँ प्रदान करता है। सिंटैक्स हाइलाइटिंग और कोड पूर्णता से लेकर स्मार्ट इंडेंटेशन और उन्नत टर्मिनल तक, Aura Text यह सुनिश्चित करता है कि आपका कोडिंग अनुभव कुशल और आनंददायक हो, साथ ही आपके PC पर अत्यंत हल्का हो।

## 🏃 शुरुआत करना

आइए अपने PC पर Aura Text सेट करें!

### पूर्वापेक्षाएँ
- Windows 10 x64 या बाद का संस्करण
- Python 3.9 या बाद का संस्करण
- Python इंस्टॉलेशन pip के साथ बूटस्ट्रैप्ड है
- (अनुशंसित) `python -m venv venv` से एक नया venv बनाएं और `venv\Scripts\activate` से सक्रिय करें
- `pip install -r requirements.txt` के माध्यम से `requirements.txt` की सामग्री स्थापित करें
- (यदि इंस्टॉलर बना रहे हैं) Inno Setup 6.4.3 या बाद का संस्करण
  
### स्थापना

आप रिलीज़ से एक पूर्वनिर्मित इंस्टॉलर डाउनलोड कर सकते हैं या खुद बना सकते हैं।

#### इंस्टॉलर बनाना

1. रिपो क्लोन करें या टारबॉल डाउनलोड करें
2. सभी पूर्वापेक्षाएँ स्थापित करें
3. प्रोग्राम को पहले कंपाइल करने के लिए `python build.py` चलाएं
4. `.iss` Inno Setup स्क्रिप्ट खोलें और Ctrl+F9 या `Build > Compile` के माध्यम से कंपाइल करें — इंस्टॉलर `Output` फ़ोल्डर में मिलेगा

##### इंस्टॉलर का उपयोग करना

बस `.exe` फ़ाइल चलाएं।

### परीक्षण

यह उन लोगों के लिए है जो मुख्यतः परीक्षण उद्देश्यों के लिए बिना इंस्टॉलेशन के चलाना चाहते हैं।
लॉगिंग के बिना प्रोग्राम चलाने के लिए `pythonw main.py` का उपयोग करें, या डीबग करने के लिए `python main.py` का उपयोग करें।

## 🛂 योगदान

योगदान ही ओपन-सोर्स समुदाय को सीखने, प्रेरित करने और बनाने के लिए एक अद्भुत स्थान बनाता है। आपके द्वारा किया गया कोई भी योगदान **बहुत सराहनीय** है।
यदि आपके पास कोई सुझाव है जो इसे बेहतर बनाए, तो कृपया रिपो को फोर्क करें और एक पुल रिक्वेस्ट बनाएं। आप "enhancement" टैग के साथ एक इश्यू भी खोल सकते हैं।
प्रोजेक्ट को स्टार देना न भूलें! फिर से धन्यवाद!

1. प्रोजेक्ट को फोर्क करें
2. अपना फीचर ब्रांच बनाएं (`git checkout -b feature/AmazingFeature`)
3. अपने परिवर्तन कमिट करें (`git commit -m 'Add some AmazingFeature'`)
4. ब्रांच पर पुश करें (`git push origin feature/AmazingFeature`)
5. एक पुल रिक्वेस्ट खोलें
